# 🎬 Movie Explorer App

## 📖 Description

Movie Explorer is a web application that allows users to search for movies and view their details using the OMDb API.

## 🎯 Purpose

The purpose of this project is to demonstrate API integration, JavaScript array methods, and UI development.

## 🔌 API Used

OMDb API - http://www.omdbapi.com/

## ✨ Features

* Search movies by name
* Display movie posters and details
* Responsive design

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript (Fetch API)

## 🚀 How to Run

1. Clone the repository
2. Open index.html in browser
3. Add your API key in script.js
